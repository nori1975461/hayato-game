# HAYATO GAME

トップビュー・シューティングゲーム。ブラウザだけで動く（ビルド不要・依存ライブラリなし）。

## あそびかた

- **矢印キー**（またはWASD） … いどう
- 武器は自動でキャラの周りを360度回転して、当たった敵を倒す
- **スペースキー** … 必殺技（ゲージ満タン時）。画面全体大爆発で雑魚全滅＋ボスに大ダメージ
- スコアで武器が**15段階**進化する：

| # | 武器 | スコア | 特徴 |
|---|---|---|---|
| 1 | ナイフ | 0 | 短い |
| 2 | 剣 | 300 | ふつう |
| 3 | 槍 | 700 | 長い |
| 4 | ダブルナイフ | 1,200 | 刃が2本に！ |
| 5 | 大剣 | 1,800 | 太い |
| 6 | 大槍 | 2,500 | 長くて太い |
| 7 | 炎の剣 | 3,300 | 火の玉発射 |
| 8 | ダブル炎の剣 | 4,200 | 火の玉×2 |
| 9 | 雷の槍 | 5,200 | 倒すと近くの敵に雷が連鎖 |
| 10 | 氷の大剣 | 6,300 | 当てた敵が凍って遅くなる |
| 11 | トリプルソード | 7,500 | 刃が3本 |
| 12 | ゴールデンソード | 9,000 | ダメージ2倍 |
| 13 | 虹の剣 | 10,500 | 虹色2本刃・ダメージ2倍 |
| 14 | ドラゴンキラー | 12,000 | ダメージ3倍＋炎 |
| 15 | エクスカリバー | 13,500 | 3本刃＋虹＋炎＋雷＋ダメージ3倍 |

- 敵：紫（+100点）／赤・速い（+150点）／緑・大型で3回斬る必要あり（+300点）
- **ボス（神話の神々）**：スコア3,000で最初のボスが咆哮とともに出現。ボス戦中は専用BGMに変わり、雑魚の2/3が逃げ出し、新しい雑魚は出現しない。倒すと+2,000点＋ハート2個＋**次のステージへ**

| ステージ | ボス | 神話 | 攻撃パターン |
|---|---|---|---|
| そうげん | ヤマタノオロチ | 日本神話 | 狙い撃ち（扇状） |
| かざん | ハデス | ギリシャ神話 | 広範囲の狙い撃ち |
| うちゅう | ゼウス | ギリシャ神話 | 黄金の雷球を全方向リング発射 |
| まかい | ロキ | 北欧神話 | 狙い撃ちとリングを交互に使うトリッキー型 |
| てんくう | オーディン | 北欧神話 | 回転しながら螺旋発射・最強HP |

- **ステージ**：そうげん→かざん→うちゅう→まかい→てんくう（背景・環境演出・BGMのキーが変わる）
- **コンボ**：連続で敵を倒すとスコア倍率アップ（+10%ずつ）。倒すたび効果音が高くなる。敵に当たるとリセット
- ライフは3つ（最大5）。敵を倒すとたまにハートが落ちて回復できる
- タイトル画面：**Cキー**＝服の色変更／**Nキー**＝名前登録（キャラの頭上に表示）／**Mキー**＝BGMのON/OFF

## 開発・確認方法

ビルド不要。`index.html` をブラウザで開くだけで動く。

## デプロイ（GitHub Pages）

1. GitHubにリポジトリ `hayato-game` を作成して push
2. リポジトリの Settings → Pages → Branch を `main` / `(root)` に設定
3. 公開URL: `https://<ユーザー名>.github.io/hayato-game/`

## ファイル構成

```
hayato-game/
├── index.html  # エントリーポイント
├── style.css   # 画面スタイル（ピクセル描画設定）
└── game.js     # ゲーム本体（スプライト・入力・更新・描画）
```
