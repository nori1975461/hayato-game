# HAYATO GAME

トップビュー・シューティングゲーム。ブラウザだけで動く（ビルド不要・依存ライブラリなし）。

## あそびかた

- **矢印キー**（またはWASD） … いどう
- 武器は自動でキャラの周りを360度回転して、当たった敵を倒す
- スコアで武器が進化する：

| 武器 | 必要スコア | 特徴 |
|---|---|---|
| ナイフ | 0 | 短い |
| 剣 | 500 | 少し長い |
| 槍 | 1,200 | 長い |
| 大剣 | 2,500 | 太くて強そう |
| 大槍 | 4,000 | 長くて太い |
| 炎の剣 | 6,000 | 剣先から火の玉を発射！ |

- 敵：紫（+100点）／赤・速い（+150点）／緑・大型で3回斬る必要あり（+300点）
- **ボス**：スコア3,000で巨大な紫ドラゴン（160px・緑の敵の5倍）が出現。超硬い・炎ブレスを扇状に吐いてくる。倒すと+2,000点＋ハート2個。倒すたびに次のボスが強くなる
- **コンボ**：連続で敵を倒すとスコア倍率アップ（+10%ずつ）。倒すたび効果音が高くなる。敵に当たるとコンボはリセット
- ライフは3つ（最大5）。敵を倒すとたまにハートが落ちて回復できる
- **Mキー** … BGMのON/OFF

## 開発・確認方法

ビルド不要。`index.html` をブラウザで開くだけで動く。

## デプロイ（GitHub Pages）

1. GitHubにリポジトリ `hayato-game` を作成して push
2. リポジトリの Settings → Pages → Branch を `main` / `(root)` に設定
3. 公開URL: `https://<ユーザー名>.github.io/hayato-game/`

## ファイル構成

```
hayato-game/
├── index.html  # エントリーポイント
├── style.css   # 画面スタイル（ピクセル描画設定）
└── game.js     # ゲーム本体（スプライト・入力・更新・描画）
```
