# HAYATO GAME

トップビュー・360度回転武器アクション。ブラウザだけで動く（ビルド不要・依存ライブラリなし）。

公開URL: https://nori1975461.github.io/hayato-game/

## あそびかた

- **矢印キー**（またはWASD） … いどう
- 武器は自動でキャラの周りを360度回転して、当たった敵を倒す
- **スペースキー** … 必殺技（ゲージ満タン時）。画面全体大爆発で雑魚全滅＋ボスに大ダメージ（盾・弱点も無視）
  - ボス戦中はゲージが自動でたまるが、**同じボス戦で使えるのは5回まで**
- **ボスに攻撃を当てるたびにポイントが入る**（ダメージ×10点）ので、ボス戦中も武器が進化していく
- **Pキー／Esc／画面右上の⏸ボタン** … 一時停止（一時停止中は画面クリックでも再開）
- タイトル画面：**Cキー**＝服の色変更／**Nキー**＝名前登録／**Mキー**＝BGMのON/OFF

- 飛び道具系（パチンコ・手裏剣・弓・爆弾・銃・火炎放射器・魔法の杖・大砲・ホーミングミサイル）は回転しながら自動で弾を発射。**遠距離武器の使用期間は長め**に配分

- ダブル攻撃（回転＋弾）は炎系・ジャベリン・インフィニティセーバーなど**少数のみ**（強すぎ防止）
- **遠距離攻撃（弾・爆発）はボスへのダメージ半分**。ボスは接近して斬りこむのが基本！（**弱点コア持ちのボスだけは例外**＝コアを弾で狙い撃つのが有効）

### ボスの攻撃

- 弾幕（狙い撃ち／広範囲／リング／螺旋）＋神様ごとに違う投擲物（炎・稲妻・剣・槍・風・トライデント・氷・ハンマー・光球・大鎌・牙・ヘビ）
- **4回に1回、10倍サイズの巨大な一撃**を放つ（赤い照準ライン→チャージ→重低音の発射音＋回転する衝撃波リング＋稲妻をまとって高速で飛んでくる。消えるときも大爆発）
- 近接攻撃：**突進パンチ／しっぽなぎはらい／ジャンプ踏みつけ（影が追ってくる）／急降下体当たり**
- ギミック：倒すと**分裂**／HP1/4で**激怒**（速く激しくなる）／**盾ガード**（バリア展開中は無効）／**弱点コア**（光る球以外はダメージなし・コアは2倍ダメージ）

## けっさん＆おみせ（5ステージごと）

5ステージクリアごとに**けっさん**：スコアが派手にカウントアップされ、**ゴールド**に変換される（スコアの1/10＋ステージボーナス）。ゴールドで**おみせ**の防具10種＋回復が買える：

てつのよろい（20%ガード）／ゆうしゃのかぶと（ライフ+1）／ちからのこて（回転+15%）／まほうのたて（弾を30%ではじく）／はやてのブーツ／かげのマント（無敵時間2倍）／ひっさつのゆびわ（ゲージ1.5倍）／いのちのおまもり（ハート2倍ドロップ）／てんしのはごろも（コンボ継続）／ふっかつのたま（1回復活）

## その他

- 敵：紫（+100点）／赤・速い（+150点）／緑・大型（+300点・ステージが進むと硬くなる）
- **コンボ**：連続撃破でスコア倍率+10%ずつ。被弾でリセット（はごろも装備中は継続）
- ライフは5つスタート（最大8、かぶとで9）。敵を倒すとたまにハートが落ちる
- ゲームオーバーしても**スペースキーで3回までコンティニュー**できる（スコア・武器・ステージ・ゴールド・装備はそのまま。ボスも残りHPのまま続行）
- ステージ20のオーディンを倒すと**ぜんステージクリア**のエンディング！

## 開発・確認方法

ビルド不要。`index.html` をブラウザで開くだけで動く。
スプライトの行長検証: `node validate-sprites.js game.js`（scratchpad参照）

## デプロイ（GitHub Pages）

`git push origin main` だけで自動反映（main / root から直接配信）。

## ファイル構成

```
hayato-game/
├── index.html  # エントリーポイント
├── style.css   # 画面スタイル（ピクセル描画設定）
└── game.js     # ゲーム本体（スプライト・入力・更新・描画）
```
