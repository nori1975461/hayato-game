# HAYATO GAME

トップビュー・シューティングゲーム。ブラウザだけで動く（ビルド不要・依存ライブラリなし）。

## あそびかた

- **矢印キー**（またはWASD） … いどう
- 武器は自動でキャラの周りを360度回転して、当たった敵を倒す
- **スペースキー** … 必殺技（ゲージ満タン時）。画面全体大爆発で雑魚全滅＋ボスに大ダメージ
- スコアで武器が**15段階**進化する：

| # | 武器 | スコア | 特徴 |
|---|---|---|---|
| 1 | ナイフ | 0 | 短い |
| 2 | 剣 | 400 | ふつう |
| 3 | 槍 | 900 | 長い |
| 4 | ダブルナイフ | 1,500 | 刃が2本に！ |
| 5 | 大剣 | 2,200 | 太い |
| 6 | 大槍 | 3,000 | 長くて太い |
| 7 | 炎の剣 | 4,000 | 火の玉発射 |
| 8 | ダブル炎の剣 | 5,200 | 火の玉×2 |
| 9 | 雷の槍 | 6,500 | 倒すと近くの敵に雷が連鎖 |
| 10 | 氷の大剣 | 8,000 | 当てた敵が凍って遅くなる |
| 11 | トリプルソード | 10,000 | 刃が3本 |
| 12 | ゴールデンソード | 12,000 | ダメージ2倍 |
| 13 | 虹の剣 | 14,500 | 虹色2本刃・ダメージ2倍 |
| 14 | ドラゴンキラー | 17,000 | ダメージ3倍＋炎 |
| 15 | エクスカリバー | 20,000 | 3本刃＋虹＋炎＋雷＋ダメージ3倍 |

- 敵：紫（+100点）／赤・速い（+150点）／緑・大型で3回斬る必要あり（+300点）
- **ボス**：スコア3,000で巨大な紫ドラゴン（翼・角・赤い目）が咆哮とともに出現。ゆっくり飛んでくる炎ブレスを避けながら戦う。出現すると雑魚の2/3は逃げ出す。倒すと+2,000点＋ハート2個＋**次のステージへ**
- **ステージ**：そうげん→かざん→うちゅう→まかい→てんくう（背景・環境演出・BGMのキーが変わる）
- **コンボ**：連続で敵を倒すとスコア倍率アップ（+10%ずつ）。倒すたび効果音が高くなる。敵に当たるとリセット
- ライフは3つ（最大5）。敵を倒すとたまにハートが落ちて回復できる
- タイトル画面：**Cキー**＝服の色変更／**Nキー**＝名前登録（キャラの頭上に表示）／**Mキー**＝BGMのON/OFF

## 開発・確認方法

ビルド不要。`index.html` をブラウザで開くだけで動く。

## デプロイ（GitHub Pages）

1. GitHubにリポジトリ `hayato-game` を作成して push
2. リポジトリの Settings → Pages → Branch を `main` / `(root)` に設定
3. 公開URL: `https://<ユーザー名>.github.io/hayato-game/`

## ファイル構成

```
hayato-game/
├── index.html  # エントリーポイント
├── style.css   # 画面スタイル（ピクセル描画設定）
└── game.js     # ゲーム本体（スプライト・入力・更新・描画）
```
